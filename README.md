## Install

```
$ yarn add @pi-cubed/graphql-proxy-server
```

## Usage

```js
import ProxyServer from '@pi-cubed/graphql-proxy-server';

const port = process.env.PORT || 5000;

new ProxyServer().start({ port }, () =>
    console.log(`Proxy server started at ${port}.`)
);
```
